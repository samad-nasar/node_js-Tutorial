NOTES:
1. if node_modules is deleted accidently, run 'npm install' to download them all.
2. if nodemon script is unable to run due to execution policy, you can change the windows execution policy to unrestricted or remoteSigned by command: 'Set-ExecutionPolicy RemoteSigned -Scope CurrentUser'
  Run this command in windows power shell.

3. to start mongoDB windows service, open cmd as administrator and run
   net start mongodb
